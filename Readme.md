# Responsive Landing Page

This project is a responsive landing page built using only HTML, CSS, and JavaScript.

The page was developed based on the provided Figma design and includes:

- Responsive layout for desktop, tablet, and mobile
- Sticky header with scroll behavior
- Image carousel with hover zoom effect
- Smooth transitions and animations
- Semantic HTML structure
- Modern CSS using Flexbox and Grid
- Vanilla JavaScript functionality without any libraries

## Project Structure

```bash
├── index.html
├── styles.css
├── script.js
├── assets/

## Features

### Sticky Header
- Appears after scrolling past the first section
- Hides when scrolling upward
- Smooth transition effects

### Image Carousel
- Interactive image slider
- Previous/Next navigation
- Thumbnail support
- Hover zoom preview

### Responsive Design
- Optimized for different screen sizes
- Mobile navigation included
- Flexible layouts using CSS Grid and Flexbox

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes
- No external frameworks or libraries were used
- Code is separated and organized for readability
- Accessibility and responsive behavior were considered during development
