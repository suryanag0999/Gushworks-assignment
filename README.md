# Responsive Landing Page

This project is a responsive landing page built using only HTML, CSS, and JavaScript based on the provided Figma design.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Sticky header with smooth scroll behavior
- Interactive image carousel
- Hover zoom preview for carousel images
- Mobile navigation menu
- Smooth animations and transitions
- Semantic HTML5 structure
- Modern CSS using Flexbox and Grid
- Clean and organized code structure
- Vanilla JavaScript implementation (no frameworks or libraries)

## Project Structure

```bash
├── index.html
├── styles.css
├── script.js
├── assets/
```


### Sticky Header
- Appears after scrolling past the first section
- Hides when scrolling upward
- Smooth transition effects

### Image Carousel
- Interactive image slider
- Previous/Next navigation
- Thumbnail support
- Hover zoom preview

### Responsive Design
- Optimized for different screen sizes
- Mobile navigation included
- Flexible layouts using CSS Grid and Flexbox

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes
- No external frameworks or libraries were used
- Code is separated and organized for readability
- Accessibility and responsive behavior were considered during development
